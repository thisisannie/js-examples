JavaScript Examples

Type the following functions into your console

Example 1: Repeat array three times

repeat([])

This function takes an array and returns it with the contents duplicated three times.

e.g.
repeat([1,2,3]) // expected output: [1,2,3,1,2,3,1,2,3]

Example 2: Reformat a string

reformat("")

This function takes a string and reformats it to have no vowels and be all lowercase exept for capitalizing the first letter.

e.g.
reformat("reFORMAT thIS BaBy") // expected output: Rfrmt ths bby

Example 3: Find the next binary number

next_binary_number([])

This function takes an array of binary digits for a number and returns the set of digits for the next number.

e.g.
next_binary_number([1,1]) // expected output: [1,0,0]
